# AI Secretary — Beginner's Guide

> This document organizes the most common issues and notes when using AI secretary.
> Recommended to read 3 minutes before first use.

---

## Most Important: Say "Wrap Up" Before Ending

AI secretary remembers things by files in your folder. If you chat and close, this chat's content **won't be organized into records**, next time it might not remember.

Build habit: **End with "wrap up"**, secretary organizes today's progress and saves.

---

## About Conversations

### Every Time You Open Is a New Conversation

Unlike LINE, Cowork **doesn't keep chat history**. Every time you open Cowork, AI starts fresh—but it reads your folder files, so still "remembers" past things.

That's why "wrap up" is critical: wrap up = save to files = next time still remembers.

### If Chatting Too Long, Open New Conversation

Chat too long (~1-2 hours intensive), AI starts compressing early conversation, might miss details.

If AI starts confusing things, just open new conversation. It re-reads records, actually clearer.

### Don't Fear New Conversations

New conversation doesn't cause amnesia. As long as you did "wrap up" before, new conversation reads complete records, seamless continuation.

---

## About Your Folder

### Always Select Same Folder

When opening Cowork, system asks which folder. **Must select same folder as first time**, else AI can't find records, like losing memory.

### Don't Hand-Delete Files in workspace/

Files in `workspace/` are AI's memory. Hand-delete = AI really forgets.

If want to clean up, ask AI to help (like "remove old to-do items"). It knows how to safely handle.

---

## Talking to AI Secretary

### Speak Like You'd Tell a Real Assistant

No special format or commands. Talk normally.

✅ "Note: next Wednesday meeting with client"
✅ "That research thing yesterday, boss said focus on skincare"
✅ "Just thought of an idea..."

### If It Gets Confused, Say So

AI isn't perfect, sometimes misremembers or misunderstands. Correct directly:

✅ "No, it's March not April"
✅ "You mixed up, Project A and B are different"

### Talk About Whatever

Work stuff, personal stuff, random thoughts, all chat-able. Secretary organizes and tracks. Doesn't need every thought to be formal "project."

---

## Common Troubleshooting

### AI Didn't Greet Me After Opening?

Might picked wrong folder. Close and reopen, confirm correct folder.

### AI Seems to Forget Last Time's Stuff?

1. Did you say "wrap up" last time? If not, that chat wasn't saved
2. Did you pick wrong folder?

### AI Got Slow or Dumb?

Chat too long. Open new conversation, let it re-read records fresh.

### AI Not Responding to My Request?

Try being more direct. Instead of "how's that thing going," try "what's the progress on beauty research."
